# 英文學習 App v5 生活3000詞庫版

本版保留 v4 的原有功能，並新增大型詞庫。

## 新增
- 3000 筆原創生活詞彙／片語／句型題庫
- 分類搜尋：飲食、購物、交通、旅遊、居家、工作、學校、健康、科技、心情、天氣、緊急等
- 情境對話練習
- 詞庫瀏覽分頁

## 保留
- 單字卡
- 真測驗：測驗時不顯示圖片提示，答完才顯示
- 錯題複習
- 新增自己的單字
- 收藏、已會進度
- 發音判斷
- PWA / GitHub Pages 可部署

## 更新方式
1. 解壓縮 ZIP
2. 上傳所有檔案到 GitHub repository 根目錄並覆蓋
3. Commit changes
4. 等 GitHub Pages 部署
5. 用 `?v=5` 開啟避免舊快取

實際內建詞庫筆數：3000
