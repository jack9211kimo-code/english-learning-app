# 英文生活單字 App 網頁版

這是一個可直接上傳到網站的英文學習 App。

## 內容

- 生活對話常用單字
- emoji 圖片記憶
- 英文發音
- 圖片測驗
- 錯題複習
- 收藏單字
- 新增自己的單字
- 匯出 / 匯入學習資料
- PWA 設定，可加入手機主畫面

## 最簡單使用方式

1. 解壓縮這個資料夾
2. 把整個資料夾內容上傳到 GitHub Pages / Netlify / Vercel
3. 用手機 Safari / Chrome 打開網址
4. iPhone：分享 → 加入主畫面
5. Android：右上角選單 → 加到主畫面

## GitHub Pages 放法

1. 建立一個 GitHub repository
2. 上傳本資料夾內所有檔案，確定 `index.html` 在最外層
3. 到 Settings → Pages
4. Source 選 Deploy from a branch
5. Branch 選 main / root
6. 等 GitHub 產生網站網址

## 注意

不要只用 iPhone「檔案」App 預覽 HTML。請放到網站後再用 Safari / Chrome 開，互動與發音會穩定很多。
