# 英文學習 App v4 發音判斷版

新增：
- 發音判斷分頁
- 使用瀏覽器語音辨識，把使用者發音轉成文字
- 與目標單字比對，給 0-100 分
- 顯示「辨識成什麼」與練習建議

限制：
- 不是專業音標/口腔位置評分
- 依賴瀏覽器 SpeechRecognition 支援
- 需要 HTTPS 和麥克風權限
- iPhone 若不穩，請用 Safari 打開 GitHub Pages 網址，並在網址後加 ?v=4

部署：
1. 解壓縮 ZIP
2. 上傳所有檔案到 GitHub repository 根目錄
3. GitHub Pages 選 main / root
4. 用 https://.../?v=4 開新版
